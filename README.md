## Setup & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/energy-hub-frontend.git
   ```

2. Open the project:
   ```bash
   cd energy-hub-frontend
   ```

3. Start with index.html or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

4. Open in browser:
   ```
   http://localhost:8000
   ```
